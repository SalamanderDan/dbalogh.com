Problem 7.28

To start we solve for angular velocity, using the forumla w = v / r.

This gives us a angular velocity of .25 radians per second.

Using this we can solve for angular distance by theta = w * t.

This gives us an angular distance of 60 radians.

With this and radius we can solve for arc length by s = r * theta.

Arc length is then 3600.
